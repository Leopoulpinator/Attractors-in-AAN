Benchmarks run on all models for several values of n

See params.txt for the parameters used, and the other files for the results:
  - out.out: complete terminal output
  - asynch & synch: results for the asynchronous/synchronous semantics:
    - nbrsolutions-*: number of solutions for each filtering mode
    - time-*: solving time (including filtering) for each filtering mode
